왁자지껄 떠들자

작성자: 베리베르비움(eyamilk)
작성일: 2015-04-01 21:07:00

덕자님이 올린 글들을 보면 허무맹랑한 이상한 글밖엔 없었습니다 ( 현재는 전부다 삭제됨, 본인이 삭제한걸로 추정 )


1. 본인이 직접 클라이언트 하나 못구해서 카페회원에게 클라이언트를 구하는걸 물어본점 ( 구글링 만으로 쉽게 구할수 있음 )

2. 클라이언트 구한 후 갑자기 개발 들어간다면서 개나소나 다아는 로그인창에서 튕기는 동영상을 올리는

쓸때없는 짓은 하면서, 절때 여지것 작업하면서 작업물에 대한 객관적인 자료를 전혀 제시할수 없었다는점

( 귀찮아서가 아니라 없어서 제시를 못하는겁니다 11시에 여러분은 속은걸 알껍니다 플레이영상 따위는 없다는거 )

3. 참고로 구동영상에 보면 ( 현재는 삭제됨 ) 바탕화면에 아이멥스, 3dp chip, 안캠코터, 다크세이버 클라이언트

이런 잡다한 프로그램밖엔 없었는데, 프로그래밍과 관련된 그 어떤 프로그램도 찾을수 없었다는점

4. 개발상황이라고 올리는글에 다크세이버 1단캐릭 시작할때 초기스텟을 물어보는 이상한질문을 했다는점

( 덕자님은 당시 예전 다크세이버랑 좀 색다르게 만들어보려고 질문했다고 댓글을 달았는데, 그럼 질문이랑 앞뒤가 안맞다는거 )

5. 테스터 뽑는다치고 계정과 비밀번호를 유저에게 직접물어보는 희안한 행동을 했다는점

( 자기가 임의의 계정을 만들어놓고 나눠주면 되는데 마치 개인정보를 수집하는거와 흡사 행동했다는거 )

6. 확성기를 개발한다는 점

( 잡담이란 확성기가 있는데, 다크세이버 유저가 맞냐고 물어볼정도로 다크세이버에 아는게 없다는거 )

7. 8단까지 구현했다는 글을 올린적이 있는데, 이미 10단까지 만들어놨던 클라이언트를 ( 10단은 미완성단계 )

도대체 8단까지 뭘 구현했는지에 대해 아무것도 언급하지 않다는점

8. 게다가 덕자님이 말한 8단퀘스트가 기존 8단 문장퀘스트 + 본인이 추가한다고한 2가지 퀘스트가 있는데

NPC에게 말걸기 + 특정 필드 클리어 2회 라는 별 이상한 퀘스트를 추가한다고 했다는점

9. 수상함을 느낀 다른 회원이 로그인이 되는 플레이영상 만이라도 공개해달라고 하자 기분나빠하면서

다음주 수요일날 전부다 공개한다며 계속 발뺌했다는 점

10. 카오스 가문사건때와 별반 차이점이 하나도없다는거 ( 객관적인 자료가 단, 1개도 없었음 )

11. 11시에 공개한다는 '스크린샷과 동영상팩' 은 도대체 뭐냐는거 ( 이글도 삭제됨 )

12. 일이 커지자 본인의 글을 전부다 삭제했다는점

13. 카페 운영진에게도 카톡계정까지만 공개가 되고, 통화는 죽어도 안된다는 점

마지막으로 한마디 하겠습니다.

이 카페는 정말 다크세이버가 하고싶어서, 만들어진 카페입니다 단순히 추억팔이나 하려고 만든 카페가 아닙니다.

여기 카페장님은 아무런 이익없이도 직접 엠게임 본사에 3번이나 찾아가고 꾸준히 노력하고 발로뛰는 사람입니다.

게다가 여기 회원님들은 카페 활동은 안하셔도 혹시나 하는마음에 계속 방문하는 사람들이 대다수입니다

그만큼 애정을 갖고있는 사람들에게 이런 장난 치는 당신 정말 못된사람입니다

11시에 서버구축에 성공했다는 단 1개의 증거만 올릴수 있다면 저는 카페를 탈퇴하겠습니다.

2시간남았습니다 두고봅시다.
